キツネたろう二世の大冒険　制作期間:3カ月　使用エンジン：Unity/C#
製作人数：プログラマー6人  モデラー・モーション 5人 企画＆UIデザイナー 1人
プログラムを担当した箇所
・草原ステージの風車前のギミック部屋
・工場ステージの扉前のギミック部屋
・ラストステージのマグマギミックの部屋
・ラストステージの倉庫番の模倣のギミック部屋
サウンドはフリー素材をお借りしています。

今回のチーム制作で初めてUnityで3Dゲームを制作しました。
工場ステージの扉前のギミック部屋で使用したボタンを押したところから
緑のラインが出る処理をボタンを押した場所をリストに記憶させて次にプレイヤーが
押した場所まで線をつなげるという処理を実装しました。
先生方にアドバイスをいただきながら制作した箇所になります。
